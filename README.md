# Healthcare Fraud Detection Research

**Researcher**: P. Chandrasekaran

## Goal
Hybrid LSTM + Isolation Forest model for healthcare fraud detection

## Results
- Baseline RF: F1=0.86, AUROC=0.97
- Target: F1>0.90, AUROC>0.98

## Innovation
- Temporal patterns (LSTM)
- Financial anomalies (Isolation Forest)
- Banking domain expertise (40+ years)
